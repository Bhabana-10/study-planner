# 📚 Deadline-Based Study Planner

A full-stack web application that helps students plan their study schedule based on deadlines, difficulty, and available daily study time.

---

## 🚀 Features

- Add, edit, and delete subjects
- Track deadlines
- Mark subjects as completed
- Generate daily study schedule
- Smart scheduling based on difficulty (Easy/Medium/Hard)

---

## 🛠 Tech Stack

- Backend: Spring Boot (Java)
- Database: MySQL
- Frontend: HTML, CSS, JavaScript
- API: REST APIs

---

## 📂 Project Structure

src/main/java → Backend (Controller, Model, Repository)  
src/main/resources/static → Frontend (HTML, CSS, JS)

---

## ▶️ How to Run the Project

1. Clone the repository  
2. Open the project in VS Code  
3. Run the backend:
   ```bash
   mvnw spring-boot:run
4. Open in browser :- http://localhost:8080
